# ChromaPro Color Picker

![ChromaPro Banner](landing/og-image.png)

**ChromaPro** is a world-class, Rank #1 optimized Chrome Extension designed for professional designers and web developers. It provides high-precision color sampling, instant palette generation, and WCAG accessibility checking in a stunning glassmorphic UI.

---

## 🚀 Features

- **Precision EyeDropper**: Sample colors from any pixel on your screen.
- **Smart Palette Generator**: Instantly generate Monochromatic (Tints), Analogous, and Complementary palettes.
- **Accessibility Checker**: Real-time contrast checks indicating if white or black text should be used.
- **One-Tap Copy**: Click any color or value (HEX, RGB, HSL) to copy it directly to your clipboard.
- **Persistent History**: Automatically saves your last 12 picked colors.
- **Zero Dependencies**: Pure Vanilla JS and CSS for lightning-fast performance.

---

## 💻 Tech Stack

- **Extension Core**: Manifest V3
- **Logic**: Vanilla JavaScript
- **Styling**: Vanilla CSS (CSS Variables, Glassmorphism)
- **APIs Used**: `EyeDropper API`, `chrome.storage.local`

---

## 🛠️ Installation (Developer Mode)

1. Clone or download this repository.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle switch in the top right corner.
4. Click the **Load unpacked** button.
5. Select the `chroma-pro-color-picker` directory.
6. The extension is now installed! Pin it to your toolbar for quick access.
